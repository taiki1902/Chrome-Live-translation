# Helium Live Translator

Helium / Chromium の現在のタブ音声を取得し、**PC内で文字起こしと翻訳を実行して字幕を重ねる** Manifest V3 拡張機能です。

YouTube・ライブ配信・学習動画などで、投稿者が字幕を用意していない場合でも、再生中の音声そのものから字幕を生成します。

## 特徴

- OpenAI APIキー不要
- 音声・認識結果・翻訳文をクラウドAPIへ送信しない
- YouTube側の字幕トラックに依存しない
- WebGPU対応GPUではGPU推論、非対応環境ではWASM/CPUへフォールバック
- Whisper ONNXによる多言語音声認識
- Chrome Translator APIが使える環境では内蔵翻訳を優先
- Heliumなど内蔵翻訳が使えない環境ではローカルOPUS-MTへ切り替え
- 翻訳字幕と認識結果の同時表示
- YouTubeのページ内フルスクリーンとブラウザフルスクリーンに対応
- 音声区間の重複除去、無音判定、遅延時のキュー制御

## 対応する字幕言語

現在のローカル版では、出力字幕は次の2言語です。

- 日本語
- 英語

日本語字幕では、英語音声はそのまま文字起こしして英日翻訳します。英語以外の音声はWhisperで一度英語へ音声翻訳し、その後、日本語へローカル翻訳します。

## インストール

1. Releasesから `helium-live-translator-v0.1.0.zip` を取得します。
2. ZIPを展開します。
3. Heliumで `chrome://extensions/` を開きます。
4. 「デベロッパーモード」を有効にします。
5. 「パッケージ化されていない拡張機能を読み込む」を押します。
6. 展開したフォルダーを選択します。

## 使い方

1. YouTubeなどで動画または配信を再生します。
2. 拡張機能を開き、音声言語と字幕言語を選択します。
3. 「翻訳を開始」を押します。
4. 初回のみ音声認識・翻訳モデルがダウンロードされます。
5. モデル準備完了後、ページ上に字幕が表示されます。

`Alt + Shift + L` でも開始・停止できます。

## ローカルモデル

| 用途           | モデル                                                             |
| -------------- | ------------------------------------------------------------------ |
| 音声認識       | `onnx-community/whisper-tiny` または `onnx-community/whisper-base` |
| 英日翻訳       | `Xenova/opus-mt-en-jap`                                            |
| 推論ランタイム | Transformers.js + ONNX Runtime Web                                 |

モデルファイルは初回利用時にHugging Faceから取得され、ブラウザのキャッシュへ保存されます。モデル取得以外で音声や字幕内容を外部へ送信しません。

## Heliumについて

HeliumはChromiumのLive Captionを無効化しているため、この拡張機能はHeliumの自動字幕UIを流用しません。`tabCapture`でユーザーが選択したタブ音声を取得し、拡張機能内のローカルモデルで独立して処理します。

## 権限

| 権限                   | 用途                                         |
| ---------------------- | -------------------------------------------- |
| `activeTab`            | ユーザーが開始した現在のタブだけを対象にする |
| `tabCapture`           | 再生中のタブ音声を取得する                   |
| `offscreen`            | 音声処理とローカル推論を継続する             |
| `scripting`            | 字幕オーバーレイを対象タブへ挿入する         |
| `storage`              | 言語・字幕表示・モデル設定を保存する         |
| Hugging Face関連ホスト | 初回のモデルファイル取得                     |

## 制限

- 初回モデル取得には数百MB以上の通信と空き容量が必要になる場合があります。
- BaseモデルはTinyより高精度ですが、GPU/RAM使用量と待ち時間が増えます。
- DRM保護コンテンツやブラウザがタブ音声取得を拒否するページでは動作しない場合があります。
- 端末性能が不足すると字幕が遅れ、古い未処理区間を自動的に破棄します。
- 完全な同時通訳ではなく、既定では約6秒単位の準リアルタイム処理です。

## 開発

```bash
npm install
npm run check
npm run build
```

ビルド済み拡張機能は `dist/extension`、配布ZIPは `dist/helium-live-translator-v0.1.0.zip` に生成されます。
